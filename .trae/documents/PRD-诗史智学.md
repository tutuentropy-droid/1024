## 1. 产品概述

"诗史智学"是一款以中国诗词为载体，串联中华历史知识的AI学习系统。通过"以诗证史、以史解诗"的创新方式，让用户在品鉴经典诗词的同时，深入了解诗词背后的历史背景、政治制度、经济形态与社会风貌。

- **核心目标**：打破诗词与历史的学科壁垒，提供沉浸式、互动式的跨学科学习体验
- **目标用户**：学生群体、诗词爱好者、历史文化学习者
- **产品价值**：让学习变得有趣高效，实现"读一首诗，懂一段史"的深度知识获取

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地存储学习进度 | 浏览时间轴、学习诗词卡片、参与智能测试、查看学习记录 |

### 2.2 功能模块

1. **首页导航**：系统介绍、功能入口、学习进度概览
2. **时间轴地图**：纵向朝代时间轴，展示各朝代诗词与历史节点
3. **学习卡片**：翻转动画卡片，正面诗词名句，背面历史解读
4. **智能测试**：根据学习进度随机出题，即时反馈答案解析
5. **学习记录**：本地存储已学内容和测试成绩

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | Hero区域 | 系统名称、Slogan、卷轴风格入场动画 |
| 首页 | 功能导航 | 三个主要功能入口卡片，悬停动效 |
| 首页 | 学习概览 | 展示已学朝代数、掌握诗词数、正确率统计 |
| 时间轴页面 | 朝代时间轴 | 纵向可滚动时间轴，每个朝代为一个节点 |
| 时间轴页面 | 朝代详情面板 | 展示朝代基本信息、代表诗人、关键历史事件 |
| 时间轴页面 | 诗词列表 | 该朝代下的所有诗词卡片入口 |
| 学习卡片页面 | 卡片展示区 | 单张卡片居中展示，支持左右切换 |
| 学习卡片页面 | 翻转交互 | 点击卡片翻转，3D翻转动画效果 |
| 学习卡片页面 | 标记功能 | 标记已学习、收藏重点卡片 |

| 智能测试页面 | 答题反馈 | 即时显示对错，展示答案解析 |
| 智能测试页面 | 成绩统计 | 本轮测试正确率、用时统计 |

## 4. 用户界面设计

### 4.1 设计风格

**整体风格**：中国古典书卷风 + 现代简约设计

- **主色调**：
  - 宣纸米白 `#F5F0E6` (背景)
  - 墨玉黑 `#2C2C2C` (文字主色)
  - 朱砂红 `#C41E3A` (强调色、标记)
  - 青花蓝 `#2B5C8A` (链接、交互元素)
  - 松烟灰 `#8B8680` (辅助文字)

- **字体**：
  - 标题："Ma Shan Zheng"（马善政楷体）或 "ZCOOL XiaoWei"（站酷小薇）
  - 正文："Noto Serif SC"（思源宋体）
  - 诗词内容：使用楷体或宋体，竖排可选

- **按钮样式**：
  - 圆角矩形，轻微内阴影
  - 悬停时：背景色加深 + 微妙上浮动画
  - 点击时：按压反馈效果

- **布局风格**：
  - 卷轴式纵向滚动
  - 卡片式内容容器
  - 留白充足，体现中式美学"虚实相生"
  - 装饰元素：云纹、回字纹、水墨笔触

- **图标风格**：
  - 线性图标，融入毛笔笔锋质感
  - 使用中国传统元素印章、卷轴、毛笔、古籍等

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | Hero区域 | 卷轴展开动画、毛笔字标题、水墨背景纹理 |
| 首页 | 功能导航 | 三张装饰性卡片，分别配"时间轴"、"翻书"、"试题"图标 |
| 时间轴页面 | 朝代时间轴 | 垂直线条连接各朝代节点，节点为印章样式 |
| 时间轴页面 | 朝代面板 | 古籍书页样式，左右分栏展示诗人与事件 |
| 学习卡片页面 | 翻转卡片 | 3D透视效果，正面米白背景配诗句，背面淡黄配历史解读 |
| 智能测试页面 | 答题区 | 题目卡片，选项为按钮式，正确显示绿色边框，错误显示红色抖动 |

### 4.3 响应式设计

- **设计原则**：Desktop-first，移动端自适应
- **断点设置**：
  - 桌面端：>1024px，完整三栏布局
  - 平板端：768px-1024px，两栏布局
  - 移动端：<768px，单栏堆叠布局
- **触控优化**：移动端按钮最小44x44px，卡片点击区域放大

### 4.4 动效设计

- **入场动画**：首页卷轴从中间向两边展开，文字逐字显现
- **卡片翻转**：CSS 3D transform实现180度Y轴旋转，0.6s缓动
- **时间轴滚动**：节点随滚动渐入，连接线从下往上绘制
- **答题反馈**：正确答案绿色闪光，错误答案红色轻微抖动
- **页面切换**：淡入淡出 + 轻微位移过渡
